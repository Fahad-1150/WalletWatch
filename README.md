Database setup

create database walletwatch;

CREATE TABLE user (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    password VARCHAR(50) NOT NULL
);

🔘 Category Navigation – Sidebar buttons to view and manage category-wise data.
